```text
If you tried LanceDB Viewer, share three details:
1. What kind of LanceDB data are you inspecting?
```

## Release Habit

For each new release:

- keep the first paragraph user-facing
- include the 30-second start steps
- include current limits
- ask the 3 feedback questions again
